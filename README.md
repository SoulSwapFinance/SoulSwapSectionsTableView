# SoulSwapSectionsTableView
 
Simple solution for static or paginating table views.

## Installation

### Swift Package Manager

```swift
dependencies: [
    .package(url: "https://github.com/SoulSwapFinance/SoulSwapSectionsTableView.git", .upToNextMajor(from: "1.0.0"))
]
```
